I. Найти численное решение уравнения $y'(x) = f(y)$, на отрезке $[0; 1]$ с шагом $h = 0.1$, где $f(y) = x * y$, $y(0) = 1$. Сравнить численное и точное решение, нарисовать графики. По методу Рунге оценить точность в каждой точке для обоих методов для случая, когда точное решение известно и когда точное решение неизвестно. Применить следующие методы:
1. Аппроксимировать y' схемой "разность вперед" (метод Эйлера);
2. Метод Рунге-Кутты:
$$k_1 = hf(x, y)$$
$$k_2 = hf(x + \frac{h}{2}, y + \frac{k_1}{2}$$
$$k_3 = hf(x + h, y + 2k_2 - k_1)$$
$$y_{n+1} = y_n + \frac{1}{6}(k_1 + 4k_2 + k_3)$$
II. Найти численное решение линейного уравнения переноса $u_t + a * u_x = 0$ при $a = 1$ с помощью схемы Годунова (разность назад/правый нижний уголок) и неявной симметричной разностной схемы на отрезке $[0, 10]$. Начальные данные: $u(x, 0) = 5$, если $x <= 3$ и $u(x, 0) = 1$, если $x > 3$. Шаг по пространству можно взять равным $\frac{1}{10}$. Провести расчеты при $r = a * \frac{\tau}{h} = {0.25, 0.5, 1, 1.25}$ на два момента времени $T = {1; 5}$. Для каждого $r$ построить графики решения на моменты времени $T = {0; 1; 5}$ (т.е. всего должно получиться 4 рисунка для каждой схемы, на каждом рисунке будут 3 графика на моменты времени $T = {0; 1; 5}$). Для каждого момента времени отрисовать точное решение.
